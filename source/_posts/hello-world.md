欢迎来到我的 Hexo 博客！这是第一篇文章。

## 快速开始

### 创建新文章

```bash
hexo new "文章标题"
```

### 本地预览

```bash
hexo server
```

访问 `http://localhost:4000` 即可预览。

### 生成静态文件

```bash
hexo generate
```

### 部署

```bash
hexo deploy
```

更多内容请参考 [Hexo 官方文档](https://hexo.io/zh-cn/docs/)。
